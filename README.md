# 📉 Layoff Data Analysis (SQL)

## 🔍 Overview
This project analyzes global layoff data using SQL to identify trends, patterns, and key insights across companies, industries, and time periods.

## 🛠 Tools Used
- SQL (MySQL)
- Data Cleaning
- Data Analysis

## 📌 Key Analysis Performed
- Total layoffs across companies and industries
- Year-wise and month-wise layoff trends
- Top companies with highest layoffs
- Industry-wise impact analysis
- Country-wise layoffs distribution

## 📂 Files Included
- layoff_analysis.sql
- dataset.csv

## 📊 Key Insights
- Consumer and retail industries contributed  significantly to total layoffs
- Layoffs peaked during the year 2022
- The most number of layoffs are from United States.

## 🚀 What I Learned
- Data cleaning using SQL
- Writing analytical queries using GROUP BY and ORDER BY
- Extracting business insights from real-world datasets
